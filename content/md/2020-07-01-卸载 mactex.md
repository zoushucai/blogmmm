# 卸载 mactex

参考：http://www.tug.org/mactex/uninstalling.html

MacTeX是mac下的latex工具，卸载方法：

1、`/usr/local/texlive/2018/` 目录，这是占用空间最大的目录

2、卸载GUI，到`/Application/` 目录里删掉几个app即可。
3、`/Library/TeX`

4、`~/Library/texlive/2019`



```shell
cd /usr/local/
sudo rm -rf texlive/

cd /Library/
sudo rm -rf TeX/

cd ~/Library/texlive
sudo rm -rf 2018
```
